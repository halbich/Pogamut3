Before using any of the export-xxx.bat files, make sure that "output" directory
exists within this directory.

export-one.bat

  Exports level geometry of one specific UT2004 map.
  
  USAGE: export-one.bat <path-to-ut2004-home-dir> <filename-of-the-ut2004-map>
  
  EXAMPLE: export-one.bat D:\Games\UT2004 DM-1on1-TrainingDay.ut2

export-all.bat

  Exports level geometry for ALL UT2004 maps of specified UT2004 installation.
  
  USAGE: export-all.bat <path-to-ut2004-home-dir>
  
  EXAMPLE: export-all.bat D:\Games\UT2004
