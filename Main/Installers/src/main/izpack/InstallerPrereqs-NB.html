!!! WARNING !!!

It is required that you RUN installer with ADMIN PRIVILEGES having UAC (User Access Control) TURNED OFF (Windows Vista / Windows 7). Otherwise the installer will fail.

We need this privileges for successful installation of NetBeans plugins and for permissions to set environment variables in order to ensure correct Apache Maven execution.

You must also have Java JDK >= 1.6 INSTALLED! Download and install JDK from http://www.oracle.com/technetwork/java/javase/downloads/index.html

If you wish to INSTALL NetBeans PLUGINS, use only NetBeans version 7.1, 7.1.1 or 7.1.2.

Once you run NetBeans after you install Pogamut 3 platform (which will trigger Pogamut 3 plugins to load for the first time inside NetBeans) you can set UAC back.
