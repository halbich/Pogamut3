It is required that you RUN installer with ADMIN PRIVILEGES having UAC (User Access Control) TURNED OFF (Windows Vista / Windows 7). Otherwise the installer will fail.

We need this privileges for permissions to set environment variables in order to ensure correct Apache Maven execution.

You must also have Java JDK >= 1.6 INSTALLED! Download and install JDK from http://www.oracle.com/technetwork/java/javase/downloads/index.html
