IMPORTANT - Windows Vista 32/64bit issue: Sometimes Pogamut plugin fails to install correctly to NetBeans 6.8 on Windows Vista. What to do?
  1. Be sure to disable User Account Control as NetBeans can't install plugins run correctly with UAC enabled (see NetBeans FAQ (http://wiki.netbeans.org/FaqInstallPluginVista)) and reinstall Pogamut
  2. If it fails, try to reinstall NetBeans, update them and launch Pogamut 3 installer again. If it fails again, try to select different NetBeans directory - choose a .netbeans directory from your user directory (e.g. c:\users\jimmy\.netbeans). DO NOT USE BROWSE but copy-paste this location (via Ctrl+C / Ctrl+V) and proceed with the installation.
  3. If it still fails to install Pogamut plugin - please contact us at pogamut@artemis.ms.mff.cuni.cz !
