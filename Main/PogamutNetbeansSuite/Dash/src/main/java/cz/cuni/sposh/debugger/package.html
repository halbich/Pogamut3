In this package are actions that are plugged into NetBeans as menu or toolbar actions. Be careful when renaming them or changing annotations, although it is supposed to be more easy than old way using layers.xml, it is still prone to breaking.

UI actions

  • DebugLapAction – action for opening the TopComponent that displayes all debugged projects and engines in each of the projects
  • WatchLapAction – state button in debug toolbar, when pressed, it listens for new engines and opens Dash windows for them
